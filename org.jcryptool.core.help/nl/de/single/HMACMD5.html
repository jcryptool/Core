HMacMD5

Ein Hash-basierter Message Authentication Code (HMAC) ist ein MAC, dessen Konstruktion auf einer kryptografischen Hash-Funktion (beispielsweise SHA oder MD5) und einem geheimen Schlüssel basiert.

HMacMD5 ist ein MAC, der die Hash-Funktion MD5 benutzt. MD5 erzeugt einen Hashwert der Länge 128 bit (16 byte). MD5 ist veraltet und wird hier nur als Beispiel benutzt. Moderne Hashfunktionen (wie SHA2 und SHA3) haben Ausgabelängen von 256 bit (32 byte) und 512 bit (64 byte).

Bedienung des Dialogs

Sie haben die Wahl, ob Sie den Schlüssel direkt (manuell) eingeben wollen oder ob Sie ihn aus dem Schlüsselspeicher (Keystore) holen wollen.

1) Falls Sie den Radiobutton Manuelle Schlüsseleingabe wählen, können Sie eine von 4 Schlüssellängen nutzen. Passend dazu wird das Schlüsselfeld mit entsprechend vielen Nullen gefüllt. Statt dieses Standardwertes können Sie auch einen eigenen Hexwert für den Schlüssel eingeben. Nach einem Klick auf den Button Fertigstellen wird der entsprechende HMAC in einem neuen Hexeditor-Fenster in der Mitte von JCT ausgegeben.

Der Output (HMAC) ist immer ein Hashwert der Länge des verwendeten Hashverfahrens (hier MD5 mit 128 bit = 16 byte), also bspw. F6 5B 71 96 D6 56 AD BA 1F 45 0B 1B 28 D7 16 75. Per Klick in das Hexeditor-Fenster und Drücken von Strg-A, Strg-C können Sie den gesamten Hashwert herauskopieren. Beim Herauskopieren werden Sie gefragt, in welcher Repräsentation Sie es ins Clipboard übernehmen wollen – hier wählen Sie Hex aus.

 

2) Alternativ können Sie den Radiobutton Schlüsselspeicher wählen: Dann versucht JCT einen passenden symmetrischen Schlüssel im Keystore zu finden. Findet JCT keinen passenden Schlüssel, können Sie in einem neuen Dialog namens "Neuer symmetrischer Schlüssel" selbst einen passenden Schlüssel im Keystore anlegen (bspw. für den User "Alice Whitehead" oder besser für einen eigenen Benutzernamen). Dazu werden Sie gefragt, mit welchem Passwort dieser neue Schlüssel geschützt werden soll. Nach Klick auf Fertigstellen landen Sie wieder im Dialog "HMacMD5". Klicken Sie hier nun auch auf Fertigstellen, kommt ein weiterer Dialog, in dem Sie das gerade gesetzte Passwort für den Schlüssel eingeben müssen. Da der neue Schlüssel nun im Keystore eingetragen ist, können Sie ihn später wieder verwenden (ohne erst nochmal einen entsprechenden Schlüssel anlegen zu müssen).
