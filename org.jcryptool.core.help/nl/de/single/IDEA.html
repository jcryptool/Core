IDEA

Beschreibung

IDEA ist ein symmetrischer Verschlüsselungs-Algorithmus, der mit Blöcken von 64 Bit Länge (= 8 Bytes) arbeitet. Die Länge des Schlüssels beträgt 128 Bit. Er wurde 1990 entwickelt und patentiert. Das Patent lief 2011 aus.

IDEA wurde in PGP (Pretty Good Privacy) verwendet und als Nachfolger von DES diskutiert.

Bedienung des Dialogs

Sie können den Inhalts des aktiven Editors direkt verschlüsseln, indem Sie auf die Schaltfläche Fertigstellen klicken – damit verwenden Sie die Standardeinstellungen. Sie werden dann nach dem Passwort zum Schutz des Schlüssels gefragt: Für den voreingestellten Benutzer "Alice" ist das "1234". Das Ergebnis wird in ein neues Hexeditor-Fenster von JCT geschrieben.

Bei JCT wird die Schlüssellänge durch den gewählten Schlüssel bestimmt. Der Dialog in JCT bietet die folgenden Parameter an:

  • für die Blockverkettung: die fünf Modi EZB, CBC, OFB, CFB und CTR
  • für das Padding: die drei Schemata PKCS#5, 01-00 Padding und kein Padding.

Drücken der langen Taste Einen neuen (IDEA-)Schlüssel im Schlüsselspeicher erzeugen, öffnet eine Dialogbox namens "Neuer symmetrischer Schlüssel". Sie können einen neuen Schlüssel entweder für einen bestehenden Benutzer wie "Alice Whitehead" oder besser für einen neuen Benutzer wie Sie selbst erzeugen (indem Sie einfach einen Benutzernamen in das Benutzerfeld eingeben und "Alice" überschreiben). Sie werden gefragt, mit welchem Passwort dieser neue Schlüssel geschützt werden soll. Nachdem der neue Schlüssel in den Schlüsselspeicher gespeichert wurde, können Sie ihn später wieder verwenden (ohne vorher einen neuen Schlüssel erstellen zu müssen).

Nach dem Klicken auf Fertigstellen kehren Sie zum Dialog "IDEA — Verschlüsselung" zurück, der nun den neu erzeugten Schlüssel in der Gruppierung "Schlüsselspeicher" auflistet. Hier müssen Sie dann auch noch auf Fertigstellen klicken, damit das Ergebnis in ein neues Hexeditor-Fenster geschrieben wird.
