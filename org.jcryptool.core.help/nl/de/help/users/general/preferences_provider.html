Krypto-Anbieter

Im Dialog Präferenzen finden Sie zum einen eine Übersicht über alle derzeit in JCrypTool installierten Krypto-Anbieter und deren Versionen. Zum anderen können Sie über die Hoch- und Runter-Buttons (auf der rechten Seite) die Reihenfolge der Krypto-Anbieter festlegen. Der Anbieter an erster Stelle wird dabei automatisch der Standard-Krypto-Anbieter.

Wählen Sie in einem Krypto-Plug-in einen bestimmten Algorithmus aus, versucht JCrypTool, für diesen eine Implementierung zu finden. Dazu werden die installierten Krypto-Anbieter in der von Ihnen festgelegten Reihenfolge durchsucht, so lange, bis eine passende Implementierung gefunden wird. Verwendet wird der Krypto-Anbieter, der als erster eine Implementierung für den Algorithmus bietet. Bietet keiner eine passende Implementierung an, erhalten Sie eine Fehlermeldung.

Standard-Krypto-Anbieter ist in JCrypTool momentan der FlexiProvider. Zusätzlich ist BouncyCastle enthalten. Die BouncyCastle wird in einer zukünftigen Version von JCT zum Standard-Krypto-Anbieter.
