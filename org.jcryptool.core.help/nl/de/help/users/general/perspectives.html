Perspektiven

Perspektiven sind eine Art Container, in denen die untergeordneten Komponenten (Sichten und Editoren) von Ihnen nahezu beliebig angepasst werden können.

JCrypTool stellt Ihnen zunächst einmal zwei Perspektiven zur Verfügung: Standard und Algorithmen. Dem Einsteiger stellt sich die Standard-Perspektive in einer üblichen Weise dar –mit Menüs und Sichten. Fortgeschrittene Anwender bieten sich in der Algorithmen-Perspektive die kryptografischen Verfahren in vielen Varianten.

Initial gibt JCrypTool das Layout dieser beiden Perspektiven vor. Bei Bedarf können Sie dieses Layout aber nahezu nach Belieben anpassen und nur von Ihnen gewünschte Sichten an den von Ihnen gewünschten Positionen anzeigen. Sollten Sie einmal eine Perspektive beim Umkonfigurieren unbrauchbar gemacht haben, können Sie diese über das Menü Fenster -> Perspektive zurücksetzen wieder in ihren Ursprungszustand zurücksetzen lassen.

Beim Start von JCrypTool wird Ihnen die zuletzt geöffnete Perspektive angezeigt. Der Wechsel zwischen den beiden Perspektiven erfolgt über die beiden Buttons Standard und Algorithmen rechts oben, oder über das Menü Fenster -> Perspektive öffnen. Im Menü Fenster -> Perspektive anpassen können Sie in einem Dialog auswählen, welche Elemente (d.h. Menüeinträge und Icons der Toolbar) in der aktiven Perspektive angezeigt werden sollen. Diese Änderungen haben nur Auswirkungen auf die gerade aktive Perspektive.

Sichten sind in jeder Perspektive unabhängig voneinander vorhanden. Das Schließen oder Verschieben einer Sicht in der Standard-Perspektive hat also keinen Einfluss auf dieselbe Sicht in der Algorithmen-Perspektive. Editoren dagegen sind in jeder Perspektive identisch. Das Schließen eines Editors in der Standard-Perspektive schließt somit auch denselben Editor in der Algorithmen-Perspektive.
