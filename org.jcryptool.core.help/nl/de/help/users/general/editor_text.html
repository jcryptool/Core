Texteditor

Mit dem Texteditor können herkömmliche Textdateien angezeigt und editiert werden. Der Editor bietet die bekannten Standardfunktionalitäten wie Rückgängig und Wiederholen, Suchen und Ersetzen sowie Kopieren und Einfügen.

Über das Menü Bearbeiten -> Öffnen mit -> Hexeditor können Sie die aktuell geöffnete Datei im Hexeditor öffnen. Der Texteditor wird dabei automatisch geschlossen.

Der Texteditor verfügt über eine Statuszeile, die sich am unteren Rand des JCrypTool befindet. Sie besteht aus drei Feldern. Im linken Feld wird angezeigt, ob die Datei bearbeitet werden kann. Im mittleren Feld wird gezeigt, ob eingegebene Zeichen eingefügt oder vorhandene Zeichen überschreiben werden. Im rechten Feld wird die aktuelle Zeilen- und Zeichennummer angezeigt.
