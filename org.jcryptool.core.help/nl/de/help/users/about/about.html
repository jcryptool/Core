Über JCrypTool

JCrypTool ist eine moderne und umfangreich erweiterbare E-Learning-Plattform rund um die Themen Kryptografie, Kryptoanalyse und IT-Security. Es unterstützt Lernende und Lehrende, Entwickler und andere an der Kryptografie Interessierte in der Anwendung und bei der Analyse von kryptografischen Mechanismen mit einer modernen und einfach zu verwendenden Plattform. JCrypTool wird als freie, Plattform-unabhängige Open-Source-Software zur Verfügung gestellt und ist Teil des CrypTool Open-Source-Projekts.

JCrypTool ist unter der Eclipse Public License (EPL) Version 1.0 lizensiert.

Mehr Informationen und Unterstützung zur Kryptografie E-Learning Plattform finden Sie unter auf unserem GitHub Projekt.

Der Source Code aller JCrypTool Core Plug-ins ist im Core Projekt verfügbar. Den Source Code der meisten JCrypTool Crypto Plug-ins finden Sie im Crypto Projekt.
