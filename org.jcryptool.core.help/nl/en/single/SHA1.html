SHA-1

Description

SHA-1 (Secure Hash Algorithm 1) is a cryptographic hash function which takes an input (message) and produces a 160-bit (20-byte) hash value (also called a message digest). The output is typically rendered as a hexadecimal number, 40 digits long.

SHA-1 was designed by the United States National Security Agency, and is a U.S. Federal Information Processing Standard. NIST formally deprecated the use of SHA-1 in 2011 and disallowed its use for digital signatures in 2013.

Like other hash functions, SHA-1 can also be used to build a random number generator.

Operation of the dialog

In this dialog you can specify how long the random number sequence to be generated is (output size) and in which format the output should be (filter).

You can set the output format with three radio buttons:

  • Binary output
  • Characters from an adjustable alphabet (default are hex characters)
  • Numbers in an adjustable value range (you can optionally still apply padding).
