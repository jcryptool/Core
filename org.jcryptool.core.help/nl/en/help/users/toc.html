User Guide

JCrypTool (JCT) is a free and open-source cryptographic e-learning software with many visualizations.

It is based on the Eclipse Rich Client Platform (RCP). You will recognize a lot of concepts in case you ever worked with Eclipse or an Eclipse RCP.

First steps and help

Beginner or not, the User Guide will introduce you in the many possibilities you have with JCrypTool. The following pages therefore contain information about the different Editors, Views and Perspectives. And of course the crypto plug-ins, starting from Algorithms, Cryptanalysis, Games, and last but not least Visualizations.

The online help (User Guide) for JCT is available in English and German. The content of the User Guide is delivered by the developers of the crypto plug-ins. This means that one plug-in may contain more help, the other one less. The JCT core team tries to ensure a minimum and consistent quality over all plug-ins and their according online help.

The user guide shows different categories, which structure the online help in JCT, e.g. General Information or Cryptology or Visuals. These categories can be opened by clicking on the plus icon in front the according entry. So you can navigate down to the help page you are looking for.

Please start reading with the help file General Information, which contains screenshots with the JCT window components, and explains how to start the online help and where to find icons with important functions.

Contact
