The calls for the algorithms MD5 and SHA-256 calculate the hash value of the content of the active editor and write the hash value then directly into a new hexeditor window (without further user input). The dialog for the three SHA-3 candidates offers to generate or verify hash values.

Further hash algorithms can be found in the Algorithm Perspective of JCT. See also the page General Information.
