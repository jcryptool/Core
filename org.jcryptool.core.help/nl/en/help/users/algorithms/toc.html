Algorithms

The JCT main menu Algorithms (in the Default Perspective) has the following entries:

Asymmetric Asymmetric methods for encryption (and signatures)
Classic Classic cipher methods
Hash Hash methods
MAC Message authentication codes
Random Number Generator Pseudo random number generator (PRNG)
Signature Signature methods
Symmetric Modern symmetric ciphers
Keystore JCT keystore (from the Algorithm Perspective)


The JCT main menu Algorithms contains a few selected cryptographic algorithms to be executed directly via a dialog box. The keystore used is the same as the one being used in the Algorithm Perspective.

Cryptographic algorithms are mainly used for encryption. The modern ones are distingished between symmetric and asymmetric algorithms. With hashes and modern symmetric ciphers MACs (message authentication codes) can be constructed; with modern asymmetric ciphers signatures can be build, and with both, cipher and hash algorithms, random number generator (PRNG) can be constructed.

Many more cryptographic algorithms in JCT can be found at

  • Visuals main menu – Visualizations of cryptographic methods
  • Algorithm Perspective – Offers the algorithms from the crypto providers Bouncy Castle and FlexiProvider (only modern algorithms)


Remark

A tabular overview about all cryptographic functions in all CrypTool programs including JCT can be found at the CrypTool website in the function overview. Setting the check there only for JCT, shows that JCT contains more than 100 function groups and more than 200 single functions (paths).
