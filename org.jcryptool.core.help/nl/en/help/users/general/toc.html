Getting online help

There are two kinds of online help incorporated in JCT: a context-sensitive help shown within the JCT window, and an exhaustive online help shown in an extra browser window.


Context-sensitive help

In oder to make using JCrypTool as easy as possible, the main window in both perspectives shows a view with context-sensitive help in the lower left corner. Here you get information about the active (currently selected) GUI part.

This context help can – under Windows and macOS – also be accessed via pressing F1) on the keyboard.


Exhaustive online help

The exhaustive online help is shown in an extra browser window. You can call this via the help icon (presented by ?) in the main tool bar at the top.

Both, context help and exhaustive online help, are part the particular plug-in. In case the plug-in author did not deliver this information (yet), general information is shown.
