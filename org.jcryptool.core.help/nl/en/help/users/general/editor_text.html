Texteditor

The Texteditor can be used to view and edit text files. The editor offers the known standard operations like undo and redo, search and replace as well as copy and paste.

Use the menu Edit -> Open with -> Hexeditor to open the currently opened file in the Hexeditor. The Texteditor will be closed automatically.

The text editor has a status bar which is located at the bottom of the JCrypTool. It consists of three fields. The left field indicates whether the file can be edited. The middle field shows whether characters entered will be inserted or existing characters will be overwritten. The right field shows the current line and character number.
