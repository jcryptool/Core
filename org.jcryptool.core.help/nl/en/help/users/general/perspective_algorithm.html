Algorithm Perspective

The Algorithm Perspective offers a functions-oriented access to a default crypto library included in JCrypTool. The algorithms shown in this perspective update themselves automatically when updating the crypto library. So a new crypto library version may automatically lead to new and extended wizards.

The JCrypTool menu bar from the Default Perspective is not available in this perspective. Note the shortened main menu in the Algorithms Perspective: The menus Algorithms, Analysis, Visuals, Games known from the Default Perspective are hidden here.
