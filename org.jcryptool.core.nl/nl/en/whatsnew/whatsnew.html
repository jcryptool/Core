Crypto Explorer

The Crypto Explorer is an alternative for using the menus: It shows on four bottom tabs – Algorithms, Analysis, Games, and Visualizations – all crypto plug-ins currently installed in the Default Perspective of JCrypTool. Simply double click on an entry to start the desired wizard or open the plug-in's view. More...


Cryptography Plug-ins

A lot of crypto plug-ins, dealing with Algorithms, Cryptanalysis, Games, and Visualizations, are waiting for you.
On the left, please click on Start JCT to enter JCT.


Help and Context Help

The JCrypTool help contains information about JCT core and crypto plug-ins. Additionally it provides background information about different cryptographic operations. Each plug-in also supports context sensitive help.
For a quick search through all parts of the Default Perspective just click Ctrl-3.
