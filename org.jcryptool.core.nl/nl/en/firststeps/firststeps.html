Perspectives

JCrypTool offers two perspectives: Default and Algorithm. For beginners, the Default Perspective contains all possibilities in a typical way from menus to views in order to easily perform cryptographic methods in standard variants. Advanced users can use modern cryptographic procedures in many variants within the Algorithm Perspective. To switch between both perspectives use the icons in the upper right or the menu Window -> Open Perspective. More...


Default Perspective

The Default Perspective is the document- or file-oriented main perspective, which offers easy access to all important JCrypTool functions. To open arbitrary files you can use the File Explorer view available on the left side in this perspective. The Crypto Explorer view on the right side enables you to call all cryptographic operations in an easy and fast way.

Document-oriented means to start with a file first and select the cryptographic operations you want to apply to this file. The original file remains untouched at all times, every operation creates a new, temporary, file.

Not all cryptographic operations require an open file in one of the editors: Visualizations and Games for example are automatically opened in their own view. More...
