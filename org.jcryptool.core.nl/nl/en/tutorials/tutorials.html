The plugins in the Default Perspective of JCrypTool normally contain a GUI and according detailed online help which together can be used as the starting point for a tutorial.


Algorithms

Under this main menu you can find both classic and modern crypto algorithms (some more are in the Visuals menu).


Analysis

Under this main menu you can find some cryptanalysis plug-ins (some more are in the Visuals menu).


Games

This main menu offers three crypto-related games. For instance, the Number Shark game helps pupils playfully to work with divisors and prime numbers.


Visualizations

Under the main menu Visuals you can find more than 30 visualizations which graphically prepare details of cryptographic methods.
